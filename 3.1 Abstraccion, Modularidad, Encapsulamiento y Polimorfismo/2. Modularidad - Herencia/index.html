Modularidad

Supongamos que tienes diferentes tipos de animales, como mamíferos y aves, y quieres organizar tu programa en módulos separados. Cada módulo manejará un tipo específico de animal.

La modularidad organiza el código para que sea fácil de entender y reutilizar, como crear módulos para diferentes tipos de animales.

Dicho de otra forma, digamos que tenemos un problema muy grande, lo mejor seria resolverlo por pedazos y no todo juntos, poniendo el ejemplo de que si tenemos un problema de 20 partes, es mejor dividirlo en 20 pedazos chiquitos y de esta forma resolvemos todo, caso contrario si lo queremos hacer todo de una vez, ya que si se hace de esa forma muy probablemente aparezca algun problema y nos haga perder todo el tiempo invertido.
