Callbacks

Un callback es simplemente una función que se pasa como argumento a otra función para que se ejecute después o cuando algo ocurra.

👍🏻 Sirven mucho cuando algo toma tiempo (como leer un archivo, consultar una API o esperar un evento).

Imagina que le dices a tu amigo:

“Cuando termines de cocinar 🧑🏻‍🍳🍳, llámame 📱📱”.

Ese “llámame” es el callback: tú no sabes cuándo terminará, pero confías en que te llamará al final.
