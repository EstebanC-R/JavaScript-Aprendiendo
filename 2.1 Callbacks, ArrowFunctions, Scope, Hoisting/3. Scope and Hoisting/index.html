Scope

Scope significa el lugar donde las variables existen y pueden ser usadas.

Hay tres tipos principales:

  1. Global: fuera de cualquier función.
  2. Local (de función): dentro de una función.
  3. De bloque: dentro de { }, con let o const.

Hoisting

El hoisting es cuando JavaScript “mueve” las declaraciones de variables y funciones al inicio del código, antes de ejecutarlo.

Pero atención:

  • Solo las declaraciones se mueven, no las asignaciones.
  • Afecta a var y a las funciones declaradas de forma tradicional.
